AI初创公司，因为都还没上市，我们讨论的都是它们的“估值”。而像Meta、谷歌这样已经在交易所挂牌的公司，我们既可以谈论它们实时的“市值”，也可以分析它们的“估值”是

截至2024年8月，月之暗面（Moonshot AI）在完成B轮融资后的估值达到了 33亿美元

截至2025年6月至7月，马斯克旗下人工智能公司xAI的最新估值约为1130亿美元，这是基于2025年6月xAI与社交媒体平台X合并后的整体估值，其中xAI估值约为800亿美元，X估值约330亿美元。

此外，xAI正准备新一轮融资，目标估值区间在1700亿美元至2000亿美元之间，融资有望在2025年7月或下月启动。如果这次融资成功，xAI的估值将大幅提升，为去年年初的十倍左右，显示出资本市场对其AI技术和未来潜力的高度认可

截至2025年4月，OpenAI的最新估值约为3000亿美元，这也是该公司在2025年3月底完成一轮400亿美元融资后的投后估值。该融资由软银主导，软银预计提供此次融资的75%资金，其余来自微软及其他投资机构。

OpenAI的估值已经超过了英特尔（约990亿美元）和AMD（约1661亿美元）市值总和，逼近阿里巴巴（约3140亿美元）的估值水平。此外，OpenAI预计2025年收入将达到约127亿美元，同比增长约两倍，显示出其快速增长的商业化能力

2025年3月初的报道，Anthropic 在由 Lightspeed Venture Partners 领投的E轮融资中筹集了35亿美元，使其最新估值达到了 615亿美元。

2025年7月，Meta Platforms (Facebook的母公司) 的最新市值约为 1.77万亿美元

2025年7月，谷歌母公司 Alphabet Inc. 的市值约为 2.25万亿美元
